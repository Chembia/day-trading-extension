🎯 Quick Start Guide

1. Understanding the Chart

Green candles = price went up
Red candles = price went down
Colored boxes = detected patterns

2. Viewing Patterns

• Hover over pattern in sidebar to highlight it on the chart
• Click pattern to zoom in on the chart
• Click "Read More" for a full explanation

3. Filtering Results

• Use "Filter" to show specific pattern types
• Adjust "Show Top" to limit results
• Use "Chart Only" to see chart without patterns

4. Refresh & Lookback

• Set a Refresh Time to load data up to that point
• Set Lookback Period to control how far back to look
• Click "Apply" or "Refresh" to reload the chart
